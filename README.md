# Copexit Umbrel App Store

Community app store for [Umbrel](https://umbrel.com/) by Copexit.

## Apps

### am-i.exposed

Bitcoin privacy scanner. Paste an address or transaction ID and get a privacy score with actionable findings. Runs 16 heuristics entirely client-side - API calls go to your local Umbrel mempool instance.

- **Requires:** Mempool app
- **Website:** [am-i.exposed](https://am-i.exposed)
- **Source:** [github.com/Copexit/am-i-exposed](https://github.com/Copexit/am-i-exposed)

## How to install

In your Umbrel dashboard:

1. Go to **App Store**
2. Click the **three-dot menu** (top right)
3. Click **Community App Stores**
4. Paste `https://github.com/Copexit/copexit-umbrel-app-store.git`
5. Click **Add**
